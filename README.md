Это программа змейка по предмету [Основы разработки ПО](https://cs.petrsu.ru/~kulakov/courses/develop/2023/task-git.php)

Автор: Александра Ломазина, 22107

## Функциональные возможности

1. Пользователь может управлять змейкой с помощью клавиш клавиатуры (например, стрелки).
    1. Пользователь нажимает клавиши клавиатуры, стрелки.
    2. Приложение отображает игровое поле, змейку, еду (обычно маркированную как фрукт) и счёт игрока.
2. Ещё пользователь может начать новую игру или завершить текущую.
    1. Пользователь нажимает клавиши клавиатуры, стрелки.
    2. Приложение отображает, что сли змейка съедает еду, то она становится длиннее, а счёт игрока увеличивается, если змейка врезается в стену или саму себя, то игра оканчивается.
